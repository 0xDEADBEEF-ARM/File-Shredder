Uses python &amp; core-utils for GNU / Linux.

Copy file shred.py to destination folder & run it.


     $ cd folder
     $ python ./shred.py


Warning: There is possibility for recovering shredded files when using journaling file systems.

`dd` `sync` `rm` `mv` `stat`
